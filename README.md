# Landing Page

Flexbox landing page project from TOP Foundations course.

This project is not intended to be responsive. It's sole purpose is to help me
get more familiar with HTML, flexbox layout methods, and CSS in general.

I wanted to bring a little bit of life to this project and not just copy the assignment
and try to duplicate the final desired result. I didn't want the final result to be just
another "follow a tutorial, and voila, instant-portfolio-project!" I try to stray away from
that mindset, because I don't learn that way.

I chose to title the project "The Anatomy of a Landing Page" out of pure curiousity. I've
always _heard_ the term, but wasn't really sure exactly what one was exactly. So... I did
some research, and learned a bit myself. Basically, I wanted to explain what a landing page
was by... building a landing page. The Odin Project didn't really go into detail (remember, the
project goal was to practice CSS/Flexbox, not inform on Marketing techniques) in the project
specifications, but what could it hurt to learn more about what I was building? It couldn't.
I hope others might be able to benefit from this too.
